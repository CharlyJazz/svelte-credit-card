# Svelte Credit Card

A component to render a credit card preview, useful for ecommerces.



### Prerequisites

You need svelte-loader or a rollup svelte configuration

### Installing


With NPM:

```bash
$ npm install svelte-credit-card
```

With Yarn:

```bash
$ yarn add svelte-credit-card
```
